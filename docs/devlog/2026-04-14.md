## Issues
1. The git runner was not running any actions and it kept saying it was missing `on`. It was there in the file locally but the file on the github repository was empty.

## Resolutions
1. I determined I was at fault. I am writing this all on Visual Studio Code to familiarize myself with a new IDE; however, unlike JetBrains that I typically use, this does not automatically write changes to disk. If you do not save a file, git does not see it as changed. This is why the action yaml file for the release build was not working. After realizing this, I saved it and pushed it and it worked great.
